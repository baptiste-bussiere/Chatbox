# CHATBOX NodeJS 

Réalisation d'un Chat en NodeJS avec SocketIO.

## Lancer le projet


Cloner le projet 

```bash
  git clone https://github.com/baptiste-bussiere/Chatbox.git
```


Installer les dependances

```bash
  npm install
```

Lancer le Projet

```bash
  npm run dev
```
